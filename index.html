Quiz 03.

quizzes.quiz03.numerical_integrate(func, a, b, n=5)¶

Compute definite integral of function.

Gauss–Legendre quadrature.

Parameters:
  • func (function) – Univariate function.
  • a (scalar) – Left-bounded of interval [a, b].
  • b (scalar) – Right-bounded of interval [a, b].
  • n ({1, 2, 3, 4, 5}, optional) – Number of points.
Returns:

out – Calculation result.
Return type:

float

Examples

>>> from math import exp
>>> numerical_integrate(lambda x: x, 0, 2, n=2)
2.0
>>> numerical_integrate(lambda x: x**2 - 2 * x - 1, 0, 3, n=3)
-2.999999999999999
>>> numerical_integrate(lambda x: exp(x) + 1, 0, 1, n=5)
2.7182818284583914
